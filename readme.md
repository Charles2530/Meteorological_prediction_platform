# 紫微气象（独角兽🦄）

大数据检测气象及地质数据

## 前端使用指南

​	前端使用 vite + vue3 + typescript作为基本框架进行web开发，集成ECharts库，以便在应用程序中创建各种图表和数据可视化，使用tailwind css和element plus美化网页布局,实现网页布局和样式的一致性。

- 启动命令:

```shell
cd front_end
# 安装相关依赖
npm install 
# 本地运行和云端运行二选一即可
    # 本地运行
    npm run dev
    # 云端运行
    npm run build
    npm run preview
```
